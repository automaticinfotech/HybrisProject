previewDataGenericEditorResponseHandler
service in module previewDataGenericEditorResponseHandlerModule

Description

previewDataGenericEditorResponseHandler is invoked by GenericEditor to handle POST response to the preview API. It is responsible for retrieving ticketId and setting it into the model for use in the updateCallback It must not set the smarteditComponentId into the editor so as to allow for multiple POST

Usage

previewDataGenericEditorResponseHandler(editor, response);

Parameters

ParamTypeDetails
editorObject

the editor invoking the response handler

responseObject

the response from the POST call to the server
