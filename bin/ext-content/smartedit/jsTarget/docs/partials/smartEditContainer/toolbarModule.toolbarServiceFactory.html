toolbarServiceFactory
service in module toolbarModule

Description

The toolbar service factory generates instances of the ToolbarService based on the gateway ID (toolbar-name) provided. Only one ToolbarService instance exists for each gateway ID, that is, the instance is a singleton with respect to the gateway ID.

Methods

  • getToolbarService(gatewayId)

    Returns a single instance of the ToolbarService for the given gateway identifier. If one does not exist, an instance is created and cached.

    Parameters
    ParamTypeDetails
    gatewayIdstring

    The toolbar name used for cross iframe communication (see gatewayProxy)

    Returns
