toolbar
directive in module toolbarModule

Description

Toolbar HTML mark-up that compiles into a configurable toolbar with an assigned ToolbarService for functionality.

Usage

as element:
<toolbar
       image-root="{String}"
       css-class="{String}"
       toolbar-name="{String}">
</toolbar>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
imageRootString

Root folder path for images

cssClassString

Space-separated string of CSS classes for toolbar item styling

toolbarNameString

Toolbar name used by the gateway proxy service
