translationServiceModule

Description

This module is used to configure the translate service, the filter, and the directives from the 'pascalprecht.translate' package. The configuration consists of:


- Initializing the translation map from the I18NAPIROOT constant.
- Setting the preferredLanguage to the UNDEFINED_LOCALE so that the i18nInterceptor request can replace it with the appropriate URI combined with the runtime browser locale retrieved from the languageService.getBrowserLocale, which is unaccessible at configuration time.
