DragAndDropServiceInterface
service in module dragAndDropServiceInterfaceModule

Description

Provides an abstract extensible drag and drop service. Used to manage and perform actions to either the SmartEdit application or the SmartEdit container.

This class serves as an interface and should be extended, not instantiated.
