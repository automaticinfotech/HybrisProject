ResourceWrapper
service in module restServiceFactoryModule

Description

The ResourceWrapper is service used to make REST calls to the wrapped target URI. It is created by the get method of the restServiceFactory. Each resource wrapper method returns a promise.

Methods

    Loads a unique component based on its identifier that must match the specified get parameters.
    The REST Service Factory evaluates placeholders in the URI, if any are provided, to verify if they match the search parameters.
