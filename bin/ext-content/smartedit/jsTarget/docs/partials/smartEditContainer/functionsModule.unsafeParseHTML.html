unsafeParseHTML
service in module functionsModule

Description

parses a string HTML into a queriable DOM object, preserving any JavaScript present in the HTML. Note - as this preserves the JavaScript present it must only be used on HTML strings originating from a known safe location. Failure to do so may result in an XSS vulnerability.

Usage

unsafeParseHTML(stringHTML,);

Parameters

ParamTypeDetails
stringHTML,String

the string representation of the HTML to parse
