l10n
filter in module l10nModule

Description

Filter that accepts a localized map as input and returns the value corresponding to the resolvedLocale of languageServiceModule and defaults to the first entry.

Usage

In HTML Template Binding

{{ l10n_expression | l10n }}

In JavaScript

$filter('l10n')(localizedMap)

Parameters

ParamTypeDetails
localizedMapObject

the map of language isocodes / values This class serves as an interface and should be extended, not instantiated.
