alertService
service in module alertServiceModule

Description

The alert service provides a simple wrapper to bind alerts to the $rootScope

Methods

  • pushAlerts(alerts)

    Binds alerts to the $rootScope

    The below is the sample code snippet for the alert object: 

    {
successful : true,
message : 'this is a successful alert',
closable : false
}

    Example of use: 

    alertService.pushAlerts(alerts);
    Parameters
    ParamTypeDetails
    alertsObject

    an array of objects containing information related to each alert
