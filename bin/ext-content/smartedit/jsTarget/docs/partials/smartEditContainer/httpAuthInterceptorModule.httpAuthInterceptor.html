httpAuthInterceptor
service in module httpAuthInterceptorModule

Description

Provides a way for global authentication by intercepting the requests before handing them to the server and response before is its given to the application code.

The interceptors are service factories that are registered with the $httpProvider by adding them to the $httpProvider.interceptors array. The factory is called and injected with dependencies and returns the interceptor object with contains the interceptor methods.

Methods

  • request(config)

    Interceptor method which gets called with a http config object, intercepts any request made with $http service. A call to any REST resource will be intercepted by this method, which then adds a authentication token to the request and the forwards it to the REST resource.

    Parameters
    ParamTypeDetails
    configObject
    • the http config object that holds the configuration information.

  • responseError(response)

    Interceptor method which intercepts all the failed responses and handles them based on the response code.

    This method handles the following response:

    • 401 response errors: handles authentication failures.
    • Non 400/401 response errors: handles any other errors by displaying an error message.
    Parameters
    ParamTypeDetails
    responseObject
    • the response object which contains the status code and other parameters that explain the response.
