EventService
service in module eventServiceModule

Description

The event service is used to transmit events synchronously or asynchronously. It also contains options to send events, as well as register and unregister event handlers.

Usage

EventService(defaultMode);

Parameters

ParamTypeDetails
defaultModeString

Uses constants to set event transmission. The EVENT_SERVICE_MODE_ASYNCH constant sets event transmission to asynchronous mode and the EVENT_SERVICE_MODE_SYNCH constant sets the event transmission to synchronous mode.

Methods

  • sendEvent(eventId, data)

    Send the event with data. The event is sent either synchronously or asynchronously depending on the event mode.

    Parameters
    ParamTypeDetails
    eventIdString

    The identifier of the event.

    dataString

    The event payload.

  • sendEvent(eventId, data)

    send the event with data synchronously.

    Parameters
    ParamTypeDetails
    eventIdString

    The identifier of the event.

    dataString

    The event payload.
