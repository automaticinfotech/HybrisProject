confirmationModalService
service in module confirmationModalServiceModule

Description

Service used to open a confirmation modal in which an end-user can confirm or cancel an action. A confirmation modal consists of a title, content, and an OK and cancel button. This modal may be used in any context in which a confirmation is required.

Methods

  • open(configuration)

    Uses the modalService to open a confirmation modal.

    The confirmation modal is initialized by a default i18N key as a title or by an override title passed through the input configuration object.

    Parameters
    ParamTypeDetails
    configurationObject

    Configuration for the confirmation modal

    PropertyTypeDetails
    titleString

    The override title for the confirmation modal. If a title is provided, it overrides the default title, which is an i18n key. This property is optional.

    descriptionString

    The description to be used as the content of the confirmation modal. This is the text that is displayed to the end user.

    Returns
    Promise

    A promise that is resolved when the OK button is actioned or is rejected when the Cancel button is actioned.
