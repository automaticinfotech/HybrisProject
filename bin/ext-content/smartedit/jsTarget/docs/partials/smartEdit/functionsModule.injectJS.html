injectJS
service in module functionsModule

Description

injectJS will inject script tags into html for a given set of sources.

Methods

  • execute(configuration)

    execute will extract a given set of sources from the provided configuration object and then inject each source as a JavaScript source tag and potential callbacks once all the sources are wired.

    Parameters
    ParamTypeDetails
    configurationObject
    • a given set of configurations.
    PropertyTypeDetails
    sourcesArray
    • an array of sources that needs to be added.
    callbackFunction
    • Callback to be triggered once all the sources are wired.
