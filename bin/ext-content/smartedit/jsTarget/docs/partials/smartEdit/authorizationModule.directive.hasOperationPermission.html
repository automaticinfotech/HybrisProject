hasOperationPermission
directive in module authorizationModule

Description

Authorization HTML mark-up that will remove elements from the DOM if the user does not have authorization defined by the input parameter permission key. This directive makes use of the AuthorizationService service to validate if the current user has the given permission.

Usage

as attribute
<ANY has-operation-permission
     has-operation-permission="{Type}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
has-operation-permissionType

The hybris type to check for given permission
