yTabset
directive in module tabsetModule

Description

The directive responsible for displaying and organizing tabs within a tabset. A specified number of tabs will display a tab header. If there are more tabs than the maximum number defined, the remaining tabs will be grouped in a drop-down menu with the header "More". When a user clicks on a tab header or an item from the drop-down menu, the content of the tabset changes to the body of the selected tab.

Note: The body of each tab is wrapped within a yTab directive.

Usage

as element:
<y-tabset
       model="{Object}"
       tabs-list="{Object[]}"
       tab-control="{Function}"
       num-tabs-displayed="{Number}">
</y-tabset>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
modelObject

Custom data to be passed to each tab. Neither the smartedit-tabset directive or the smartedit-tab directive can modify this value. The tabs' contents determine how to parse and use this object.

tabsListObject[]

A list that contains the configuration for each of the tabs to be displayed in the tabset.

PropertyTypeDetails
idstring

The ID used to track the tab within the tabset.

titleString

The tab header.

templateUrlString

Path to the HTML fragment to be displayed as the tab content.

hasErrorsboolean

Flag that indicates whether a visual error is to be displayed in the tab or not.

tabControlFunction

An optional parameter. A function that will be called with scope as its parameter. It allows the caller to register extra functionality to be executed in the child tabs.

numTabsDisplayedNumber

The number of tabs for which tab headers will be displayed. The remaining tab headers will be grouped within the 'MORE' drop-down menu.
