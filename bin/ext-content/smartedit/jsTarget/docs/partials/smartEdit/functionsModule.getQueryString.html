getQueryString
service in module functionsModule

Description

getQueryString will convert a given object into a query string.

Below is the code snippet for sample input and sample output:

var params = {
 key1 : 'value1',
 key2 : 'value2',
 key3 : 'value3'
 }
 
 var output = getQueryString(params);
 
 // The output is '?&key1=value1&key2=value2&key3=value3'

Usage

getQueryString(params);

Parameters

ParamTypeDetails
paramsObject

Object containing a list of params.

Returns

String

a query string
