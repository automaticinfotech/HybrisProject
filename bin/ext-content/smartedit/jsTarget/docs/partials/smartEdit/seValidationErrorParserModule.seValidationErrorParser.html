seValidationErrorParser
service in module seValidationErrorParserModule

Description

This service provides the functionality to parse validation errors received from the backend.

Methods

  • parse()

    Parses extra details, such as language and format, from a validation error message. These details are also stripped out of the final message. This function expects the message to be in the following format:

    var message = "Some validation error occurred. Language: [en]. Format: [widescreen]. SomeKey: [SomeVal]."

    The resulting error object is as follows: 

    {
    message: "Some validation error occurred."
    language: "en",
    format: "widescreen",
    somekey: "someval"
}
