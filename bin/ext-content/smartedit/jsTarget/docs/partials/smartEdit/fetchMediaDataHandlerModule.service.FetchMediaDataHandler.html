FetchMediaDataHandler
service in module fetchMediaDataHandlerModule

Description

implementation of FetchDataHandlerInterface for "Media" cmsStructureType

Methods

  • findByMask()

    implementation of FetchDataHandlerInterface.findBymask Given a field descriptor from GenericEditor and a mask this method will query a REST API and returned a filtered list of media assets with the following format:

    • id: Identifier of the media.
    • url: URL where the media is located.
    [{
	  id: 'someid1',
	  url: 'fullURLToMedia1'
  }, {
	  id: 'someid2',
	  url: 'fullURLToMedia2'
   }]
