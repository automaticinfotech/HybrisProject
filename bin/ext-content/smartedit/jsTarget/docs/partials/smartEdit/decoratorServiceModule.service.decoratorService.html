decoratorService
service in module decoratorServiceModule

Description

This service enables and disables decorators. It also maps decorators to SmartEdit component typesâ€“regardless if they are enabled or disabled.

Methods

  • addMappings(map)

    This method enables a list of decorators for a group of component types. The list to be enable is identified by a matching pattern. The list is enabled when a perspective or referenced perspective that it is bound to is activated/enabled.

    Parameters
    ParamTypeDetails
    mapObject

    A key-map value; the key is the matching pattern and the value is an array of decorator keys. The key can be an exact type, an ant-like wild card, or a full regular expression: 

    decoratorService.addMappings({
            '*Suffix': ['decorator1', 'decorator2'],
            '.*Suffix': ['decorator2', 'decorator3'],
            'MyExactType': ['decorator3', 'decorator4'],
            '^((?!Middle).)*$': ['decorator4', 'decorator5']
        	});

  • getDecoratorsForComponent(componentType)

    This method retrieves a list of decorator keys that is eligible for the specified component type. The list retrieved depends on which perspective is active.

    This method uses the list of decorators enabled by the addMappings method.

    Parameters
    ParamTypeDetails
    componentTypeString

    The component type to be decorated.

    Returns
    Promise

    It returns a list of decorator keys.
