componentHandlerService
service in module componentHandlerServiceModule

Description

The service provides convenient methods to get DOM references of smartEdit components both in the original laye rof the storefornt and in the smartEdit overlay

Methods

  • methodsOf_getAllComponentsSelector()

    Retrieves the jQuery selector matching all smartEdit components that are not of type ContentSlot

  • methodsOf_getAllSlotsSelector()

    Retrieves the jQuery selector matching all smartEdit components that are of type ContentSlot

  • methodsOf_getComponent(smarteditComponentId, smarteditComponentType, cssClass)

    Retrieves the jQuery wrapper around a smartEdit component identified by its smartEdit id, smartEdit type and an optional class This method can only be invoked from the smartEdit application and not the smartEdit container.

    Parameters
    ParamTypeDetails
    smarteditComponentIdString

    smarteditComponentTypeString

    the component type as per the smartEdit contract with the storefront

    Object
