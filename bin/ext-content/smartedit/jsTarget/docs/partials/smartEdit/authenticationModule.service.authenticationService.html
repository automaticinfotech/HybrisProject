authenticationService
service in module authenticationModule

Description

The authenticationService is used to authenticate and logout from SmartEdit. It also allows the management of entry points used to authenticate the different resources in the application.

Methods

  • isAuthEntryPoint(resource)

    Indicates if the resource URI provided is one of the registered authentication entry points.

    Parameters
    ParamTypeDetails
    resourceString

    The URI to compare

    Returns
    Boolean

    Flag that will be true if the resource URI provided is an authentication entry point.

  • isAuthenticated(resource)

    Indicates if the resource URI provided maps to a registered authentication entry point and the associated entry point has an authentication token.

    Parameters
    ParamTypeDetails
    resourceString

    The URI to compare

    Returns
    Boolean

    Flag that will be true if the resource URI provided maps to an authentication entry point which has an authentication token.

  • authenticate(resource)

    Authenticates the current SmartEdit user against the entry point assigned to the requested resource. If no suitable entry point is found, the resource will be authenticated against the DEFAULT_AUTHENTICATION_ENTRY_POINT

    Parameters
    ParamTypeDetails
    resourceString

    The URI identifying the resource to access.

    Returns
    Promise

    A promise that resolves if the authentication is successful.

  • filterEntryPoints(resource)

    Will retrieve all relevant authentication entry points for a given resource. A relevant entry point is an entry value of the authenticationMap found in sharedDataService.The key used in that map is a regular expression matching the resource. When no entry point is found, the method returns the DEFAULT_AUTHENTICATION_ENTRY_POINT

    Parameters
    ParamTypeDetails
    resourcestring

    The URL for which a relevant authentication entry point must be found.

  • logout()

    The logout method removes all stored authentication tokens and redirects to the landing page.

  • setReAuthInProgress(entryPoint)

    Used to indicate that the user is currently within a re-authentication flow for the given entry point. This flow is entered by default through authentication token expiry.

    Parameters
    ParamTypeDetails
    entryPointString

    The entry point which the user must be re-authenticated against.
