synchronizationService
service in module synchronizationServiceModule

Description

The synchronization service manages RESTful calls to the synchronization service's backend API.

Methods

  • getCatalogSyncStatus(catalog)

    This method is used to get the synchronization status of a catalog.

    Parameters
    ParamTypeDetails
    catalogObject

    An object that contains the information about the catalog to be synchronized.

  • startAutoGetSyncData(catalog, callback)

    This method triggers the auto synchronization status update for a given catalog.

    Parameters
    ParamTypeDetails
    catalogObject

    An object that contains the information about the catalog to be synchronized.

    callbackFunction

    The callback function that will be called upon catalog status update.

  • stopAutoGetSyncData(catalog)

    This method stops the auto synchronization status update for a given catalog.

    Parameters
    ParamTypeDetails
    catalogObject

    An object that contains the information about the catalog to be synchronized.

  • updateCatalogSync(catalog)

    This method is used to synchronize a catalog.

    Note: synchronization is one-way from a staged catalog version to an online catalog version.

    Parameters
    ParamTypeDetails
    catalogObject

    An object that contains the information about the catalog to be synchronized.
