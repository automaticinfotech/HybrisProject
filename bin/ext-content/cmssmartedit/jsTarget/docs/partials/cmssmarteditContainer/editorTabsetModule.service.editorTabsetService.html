editorTabsetService
service in module editorTabsetModule

Description

The Editor Tabset Service keeps track of a list of tabs and their configuration, which includes the id, title, and template URL. The service allows registering, removing, and listing tabs. By default it contains the following ones:

  • Admin Tab
  • Basic Tab
  • Generic Tab

This service is used by the editorTabset directive to determine the tabs to be displayed in the tabset.

Methods

  • deleteTab(tabId)

    Removes a tab from the list of registered tabs.

    Parameters
    ParamTypeDetails
    tabIdString

    The ID of the tab to be removed.

  • getTabsList()

    Retrieves the list of registered tabs to display in an editing tabset.

    Returns
    Array

    array containing the registered tabs to display in an editing tabset.

  • registerTab(tabId, tabTitle, tabTemplateUrl)

    This method allows registering a tab to be displayed in an editing tabset.

    Note: If the tabId is not unique, this method will override the previous tab configuration.

    Parameters
    ParamTypeDetails
    tabIdString

    An ID for the tab. It has to be unique, otherwise the previous tab configuration will be overriden.

    tabTitleString

    The string displayed as the tab header.

    tabTemplateUrlString

    Path to the HTML fragment to be displayed as the tab content.
