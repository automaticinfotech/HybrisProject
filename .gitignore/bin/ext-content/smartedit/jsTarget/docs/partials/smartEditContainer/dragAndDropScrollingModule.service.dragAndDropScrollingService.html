dragAndDropScrollingService
service in module dragAndDropScrollingModule

Description

The Drag and Drop Scrolling service manages the scrolling of a page while dragging a SmartEdit component. It creates, appends, shows and hides UI hints (regions in the page that scroll the page up or down when hovered).

This service is only intended to be used by the base SmartEdit Drag and Drop service.

Methods

  • connectWithDragAndDrop(dragAndDropConfiguration)

    This method enables the communication between this service and the base SmartEdit Drag and Drop service.

    Parameters
    ParamTypeDetails
    dragAndDropConfigurationObject

    The object used to configure SmartEdit Drag and Drop service.

  • disableScrolling(startedInExternalFrame, isExternalFrame)

    This method is executed when the dragging of a SmartEdit component stops. It hides UI hints and disables scrolling.

    Parameters
    ParamTypeDetails
    startedInExternalFrameBoolean

    Flag that indicates if the drag event was started in the outer frame.

    isExternalFrameBoolean

    Flag that indicates if the current method is called in the outer frame.

  • enableScrolling(startedInExternalFrame, isExternalFrame)

    This method is executed when the dragging of SmartEdit component is started. It shows the UI hints and enables scrolling when the cursor hovers over them.

    Parameters
    ParamTypeDetails
    startedInExternalFrameBoolean

    Flag that indicates if the drag event was started in the outer frame.

    isExternalFrameBoolean

    Flag that indicates if the current method is called in the outer frame.

  • initialize()

    This method prepares the page to allow scrolling while a SmartEdit component is dragged. It creates and appends UI hints and attaches the necessary event handlers.
