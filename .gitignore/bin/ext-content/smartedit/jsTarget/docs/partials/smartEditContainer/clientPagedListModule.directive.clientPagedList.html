clientPagedList
directive in module clientPagedListModule

Description

Directive responsible for displaying a client-side paginated list of items with custom renderers. It allows the user to search and sort the list.

Usage

as element:
<client-paged-list
       items="{Array}"
       keys="{Array}"
       renderers="{Object}"
       injected-context="{Object}"
       reversed="{Boolean}"
       items-per-page="{Number}"
       query="{Object}"
       display-count="{Boolean}">
</client-paged-list>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
itemsArray

An array of item descriptors.

keysArray

An array of object(s) with a property and an i18n key. The properties must match one at least one of the descriptors' keys and will be used as the columns of the table. The related i18n keys are used for the column headers' title.

renderersObject

An object that contains HTML renderers for specific keys property. A renderer is a function that returns a HTML string. This function has access to the current item and key.

injectedContextObject

An object that exposes values or functions to the directive. It can be used by the custom HTML renderers to bind a function to a click event for example.

reversedBoolean

If set to true, the list will be sorted descending.

itemsPerPageNumber

The number of items to display per page.

queryObject

The ngModel query object used to filter the list.

displayCountBoolean

If set to true the size of the filtered collection will be displayed.

Example

<client-paged-list items="pageListCtl.pages"
            keys="[{
                    property:'title',
                    i18n:'pagelist.headerpagetitle'
                    },{
                    property:'uid',
                    i18n:'pagelist.headerpageid'
                    },{
                    property:'typeCode',
                    i18n:'pagelist.headerpagetype'
                    },{
                    property:'template',
                    i18n:'pagelist.headerpagetemplate'
                    }]"
            renderers="pageListCtl.renderers"
            injectedContext="pageListCtl.injectedContext"
            sort-by="'title'"
            reversed="true"
            items-per-page="10"
            query="pageListCtl.query.value"
            display-count="true"
  ></paged-list>

Example of a renderers object

renderers = {
    name: function(item, key) {
        return "<a data-ng-click=\"injectedContext.onLink( item.path )\">{{ item[key.property] }}</a>";
    }
};

Example of an injectedContext object 

injectedContext = {
    onLink: hitch(this, function(link) {
        if (link) {
            var experiencePath = this._buildExperiencePath();
            iFrameManager.setCurrentLocation(link);
            $location.path(experiencePath);
        }
    })
};
