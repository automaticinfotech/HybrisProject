compileHtml
directive in module compileHtmlModule

Description

Directive responsible for evaluating and compiling HTML markup.

Usage

as attribute
<ANY compile-html
     string="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
StringString

HTML string to be evaluated and compiled.

Example

<div compile-html="<a data-ng-click=\"injectedContext.onLink( item.path )\">{{ item[key.property] }}</a>"></div>
