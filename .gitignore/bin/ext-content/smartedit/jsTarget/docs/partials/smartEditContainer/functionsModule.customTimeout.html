customTimeout
service in module functionsModule

Description

customTimeout will call the javascrit's native setTimeout method to execute a given function after a specified period of time. This method is better than using $timeout since it is difficult to assert on $timeout during end-to-end testing.

Usage

customTimeout(func, duration);

Parameters

ParamTypeDetails
funcFunction

function that needs to be executed after the specified duration.

durationNumber

time in milliseconds.
