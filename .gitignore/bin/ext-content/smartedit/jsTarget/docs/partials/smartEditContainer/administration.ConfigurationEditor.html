ConfigurationEditor
service in module administration

Description

The Configuration Editor Service is a convenience service that provides the methods to manage configurations within the Configuration Editor UI, such as filtering configurations, adding entries and removing entries.

Methods

  • addEntry()

    The Add Entry method adds an entry to the list of configurations.

  • filterConfiguration()

    Method that returns a list of configurations by filtering out only those configurations whose 'toDelete' parameter is set to false.

    Returns
    Object

    A list of filtered configurations.

  • init(loadCallback)

    The init method initializes the configuration editor and loads all the configurations so they can be edited.

    Parameters
    ParamTypeDetails
    loadCallbackFunction

    The callback to be executed after loading the configurations.

  • removeEntry(entry, configurationForm)

    The Remove Entry method deletes the specified entry from the list of configurations. The method does not delete the actual configuration, but just removes it from the array of configurations. The entry will be deleted when a user clicks the Submit button.

    Parameters
    ParamTypeDetails
    entryObject

    The object to be deleted

    configurationFormObject

    The form object which is an instance of FormController that provides methods to monitor and control the state of the form.

  • submit(configurationForm)

    The Submit method saves the list of available configurations by making a REST call to a web service. The method is called when a user clicks the Submit button in the configuration editor.

    Parameters
    ParamTypeDetails
    configurationFormObject

    The form object that is an instance of FormController. It provides methods to monitor and control the state of the form.
