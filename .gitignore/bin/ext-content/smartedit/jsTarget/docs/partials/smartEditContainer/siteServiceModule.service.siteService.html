siteService
service in module siteServiceModule

Description

The Site Service fetches all sites configured on the hybris platform using REST calls to the cmswebservices sites API.

Methods

  • getSiteById()

    Fetches a site, configured on the hybris platform, by its uid. The sites if fetched using REST calls to the cmswebservices sites API.

    Returns
    object

    a site descriptos. a descriptor provides the following site properties: uid, name, previewUrl, and redirectUrl.

  • getSites()

    Fetches a list of sites configured on the hybris platform. The list of sites if fetched using REST calls to the cmswebservices sites API.

    Returns
    Array

    An array of site descriptors. Each descriptor provides the following site properties: uid, name, previewUrl, and redirectUrl.
