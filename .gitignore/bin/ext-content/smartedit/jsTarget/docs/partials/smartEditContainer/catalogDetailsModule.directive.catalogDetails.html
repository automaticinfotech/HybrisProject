catalogDetails
directive in module catalogDetailsModule

Description

Directive responsible for displaying a catalog details

redirects to the storefront page for the given default experience (siteId, catalogId, catalogVersionId)

Usage

as element:
<catalog-details
       catalog="{String}">
</catalog-details>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
catalogString

The catalog that needs to be displayed
