iframeClickDetectionService
service in module iframeClickDetectionServiceModule

Description

The iframe Click Detection service uses the gatewayProxy service to listen for click events, specifically mousedown events, from the SmartEdit application. This service also provides the functionality to register callbacks to click events.

Methods

  • onIframeClick()

    Triggers all callbacks currently registered to the service. This function is registered as a listener through the gatewayProxy.

  • registerCallback(id, callback)

    Registers a callback to be triggered when a click occurs in the iframe.

    Parameters
    ParamTypeDetails
    idString

    The ID to register the given callback

    callbackFunction

    The callback to be triggered.

  • removeCallback(id)

    Removes a callback registered to the given ID

    Parameters
    ParamTypeDetails
    idString

    The ID of the callback to remove
