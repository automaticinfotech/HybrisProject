iframeClickDetectionService
service in module iframeClickDetectionServiceModule

Description

The iframe Click Detection service leverages the gatewayProxy service to propagate click events, specifically mousedown events, to the SmartEdit container.

Methods

  • onIframeClick()

    The proxy function that delegates calls to the SmartEdit container.

    Returns
    Promise

    Promise that is resolved when the SmartEdit container completes the callback.
