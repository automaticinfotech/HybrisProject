regExpFactory
service in module functionsModule

Description

regExpFactory will convert a given pattern into a regular expression. This method will prepend and append a string with ^ and $ respectively replaces and wildcards (*) by proper regex wildcards.

Usage

regExpFactory(pattern);

Parameters

ParamTypeDetails
patternString

any string that needs to be converted to a regular expression.

Returns

RegExp

a regular expression generated from the given string.
