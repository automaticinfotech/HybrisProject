PerspectiveServiceInterface
service in module perspectiveInterfaceModule

Methods

  • hasActivePerspective()

    This method returns true if a perspective is selected.

    Returns
    Boolean

    The key of the active perspective.

  • isEmptyPerspectiveActive()

    This method returns true if the current active perspective is the Preview mode (No active overlay).

    Returns
    Boolean

    Flag that indicates if the current perspective is the Preview mode.

  • register(configuration)

    This method registers a perspective. When an end user selects a perspective in the SmartEdit web application, all features bound to the perspective will be enabled when their respective enablingCallback functions are invoked and all features not bound to the perspective will be disabled when their respective disablingCallback functions are invoked.

    Parameters
    ParamTypeDetails
    configurationObject

    The configuration that represents the feature to be registered.

    PropertyTypeDetails
    keyString

    The key that uniquely identifies the perspective in the registry.

    nameI18nKeyString

    The i18n key that stores the perspective name to be translated.

    descriptionI18nKeyString

    The i18n key that stores the perspective description to be translated. The description is used as a tooltip in the web application. This is an optional parameter.

    featuresArray

    A list of features to be bound to the perspective.

    perspectivesArray

    A list of referenced perspectives to be bound to this system perspective. This list is optional.

  • selectDefault()

    This method switches the currently-selected perspective to the default perspective. If no value has been stored in the smartedit-perspectives cookie, the value of the default perspective is se.none. If a value is stored in the cookie, that value is used as the default perspective.

  • switchTo(key)

    This method actives a perspective identified by its key and deactivates the currently active perspective. Activating a perspective consists in activating any feature that is bound to the perspective or any feature that is bound to the perspective's referenced perspectives and deactivating any features that are not bound to the perspective or to its referenced perspectives.

    Parameters
    ParamTypeDetails
    keyString

    The key that uniquely identifies the perspective to be activated. This is the same key as the key used in the register method.
