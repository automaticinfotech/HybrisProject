parseQuery
service in module functionsModule

Description

parseQuery will convert a given query string to an object.

Below is the code snippet for sample input and sample output:

var query = '?key1=value1&key2=value2&key3=value3';
 
var output = parseQuery(query);

// The output is { key1 : 'value1', key2 : 'value2', key3 : 'value3' }

Usage

parseQuery(query);

Parameters

ParamTypeDetails
queryString

String that needs to be parsed.

Returns

Object

an object containing all params of the given query
