ToolbarServiceInterface
service in module toolbarInterfaceModule

Description

Provides an abstract extensible toolbar service. Used to manage and perform actions to either the SmartEdit application or the SmartEdit container.

This class serves as an interface and should be extended, not instantiated.

Methods

  • addItems(actions)

    Takes an array of actions and maps them internally for display and trigger through an internal callback key.

    Parameters
    ParamTypeDetails
    actionsObject[]
    • List of actions
    PropertyTypeDetails
    keyString
    • Unique identifier of the toolbar action item.
    callbackFunction
    • Callback triggered when this toolbar action item is clicked.
    i18nkeyString
    • Description translation key
    iconsString[]
    • List of image URLs for the icon images (only for ACTION and HYBRID_ACTION)
    typeString
    • TEMPLATE, ACTION, or HYBRID_ACTION
    includeString
    • HTML template URL (only for TEMPLATE and HYBRID_ACTION)
    priorityInteger
    • Determines the position of the item in the toolbar, ranging from 0-1000 with the default priority being 500.
    sectionString
    • Determines the sections(left, middle or right) of the item in the toolbar. An item with a higher priority number will be to the right of an item with a lower priority number in the toolbar.
