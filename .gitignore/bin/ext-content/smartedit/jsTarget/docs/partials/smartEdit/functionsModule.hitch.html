hitch
service in module functionsModule

Description

hitch will create a new function that will pass our desired context (scope) to the given function. This method will also pre-bind the given parameters.

Usage

hitch(scope, method);

Parameters

ParamTypeDetails
scopeObject

scope that is to be assigned.

methodFunction

the method that needs binding.

Returns

Function

a new function thats binded to the given scope
