MODAL_BUTTON_ACTIONS
object in module modalServiceModule

Description

Injectable angular constant
Defines the action to be taken after executing a button on a modal window. To be used when adding a button to the modal, either when opening a modal (see ModalManager.getButtons()) or when adding a button to an existing modal (see modalService.open())

Example: 

     myModalManager.addButton({
         id: 'button id',
         label: 'close_modal',
         action: MODAL_BUTTON_ACTIONS.CLOSE
     });

Properties

  • CLOSE

    Indicates to the ModalManager that after executing the modal button, the modal window should close, and the promise returned by the modal should be resolved.

  • DISMISS

    Indicates to the ModalManager that after executing the modal button, the modal window should close, and the promise returned by the modal should be rejected.

  • NONE

    Indicates to the ModalManager that after executing the modal button no action should be performed.
