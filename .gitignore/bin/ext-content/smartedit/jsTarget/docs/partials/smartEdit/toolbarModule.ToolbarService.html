ToolbarService
service in module toolbarModule

Description

The inner toolbar service is used to add toolbar actions that affect the inner application, publish aliases to the outer application through the proxy, and store and manage callbacks made by private key. The service is a managed singleton; the toolbarServiceFactory provides a getToolbarService function that returns a single instance of the ToolbarService for the gateway identifier, that is, the toolbar-name provided by the outer toolbar.

Uses gatewayProxy for cross iframe communication, using the toolbar name as the gateway ID.

Inherited Methods from ToolbarServiceInterface: addItems

Usage

ToolbarService(gatewayId);

Parameters

ParamTypeDetails
gatewayIdString

Toolbar name used by the gateway proxy service.
