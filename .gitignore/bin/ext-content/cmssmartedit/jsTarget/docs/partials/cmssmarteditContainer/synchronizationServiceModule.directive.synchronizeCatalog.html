synchronizeCatalog
directive in module synchronizationServiceModule

Description

The Synchronize Catalog directive is used to display the synchronization area in the landing page for each catalog and to update the status of the synchronization button.

The directive provides communication by exposing the (syncCatalog) method. When called, the directive accesses the synchronization service to synchronize a specified catalog.

Usage

as element:
<synchronize-catalog
       catalog="{Object}"
       sync-catalog="{Function}">
</synchronize-catalog>

Parameters

ParamTypeDetails
catalogObject

An object that contains the catalog details. It allows the directive to load or update the synchronization data for a specified catalog.

syncCatalogFunction

A function that triggers the synchronization of a catalog using the synchronization service. It invokes a modal pop-up to confirm the action before it calls the service layer.
